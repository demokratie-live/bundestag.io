## Getting started

Clone the git repo & run the project
```
git clone git@github.com:bundestag/bundestag.io.git
cd bundestag.io
yarn install
```

### Compile and start
```
yarn ~~start~~ dev
```

### Import Data via Scraper
```
yarn imp
```

### Diff scraped Data Versions
```
yarn diff
```

### Test Project
```
yarn lint
```
